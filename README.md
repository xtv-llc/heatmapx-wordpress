# HeatMapX – Heatmaps & A/B Testing (WordPress plugin)

Official WordPress plugin for [HeatMapX](https://heatmapx.com).
Paste your site key — heatmaps, session analytics and A/B testing start working. No theme editing.

## Development
- `composer install && composer test` — unit tests
- `composer lint` — PHPCS (WordPress-Extra)
- `npm run wp-env start` — local WordPress for E2E
